# ekNojor Client

Fast reader web app for ekNojor.

## Run

```bash
npm install
npm run dev
```

Default URL:

```txt
http://localhost:3002
```

The reader fetches published news from the admin/backend API:

```env
EKNOJOR_API_BASE_URL="http://localhost:3000"
```

Make sure the admin app is running and has published news.
